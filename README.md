# 이기운 키우기

Three.js로 만든 가벼운 미니 육성 게임입니다.

복잡한 React 앱이 아니라, 처음 opossum 프로젝트처럼 `index.html + src/main.js + src/style.css` 중심으로 동작합니다.

## 컨셉

검은 머리, 차분한 인상, 폭신한 검정 자켓, 코딩/커피/독서/음악을 좋아하는 이기운 캐릭터를 키우는 미니 게임입니다.

버튼을 누르면 아이템이 3D 캐릭터에게 날아가고, 스탯과 레벨이 올라갑니다.

## 주요 기능

- Three.js 3D 방
- 절차적으로 만든 이기운 3D 캐릭터
- 코딩하기
- 커피 마시기
- 독서하기
- 음악 듣기
- 운동하기
- 밥 먹기
- 쉬기
- 외출하기
- 방 정리하기
- 캐릭터 클릭/드래그 상호작용
- 레벨/경험치/코인
- 에너지/기분/집중/코딩/사회성
- 업적
- localStorage 저장

## 기술 스택

- Vite
- Three.js
- Vanilla JavaScript
- CSS

## 설치
